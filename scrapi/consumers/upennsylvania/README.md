UPenn-consumer

More information: https://github.com/CenterForOpenScience/SHARE/blob/master/providers/edu.upenn.md

Example API call: http://repository.upenn.edu/do/oai/?verb=ListRecords&metadataPrefix=oai_dc&from=2014-09-26
